How-To Guides

Creating a Root Certificate
  1. Go to the home page
  2. Click on "Create Root Certificate"
  3. Fill in the required information:
    • Organization Name
    • Common Name
    • Country
    • Validity Period
  4. Click "Create" to generate your root certificate

Creating Client/Server Certificates

  1. Ensure you have created a root certificate first
  2. Go to the Certificates page
  3. Click "Create New Certificate"
  4. Fill in the certificate details:
    • Certificate Type (Client or Server)
    • Common Name
    • Organization
    • Validity Period
  5. Click "Create" to generate your certificate

Managing Certificates

  • View certificate details by clicking the "View" button
  • Download certificates using the "Download" button
  • Remove certificates using the "Delete" button
  • All certificates are stored securely in the data directory
